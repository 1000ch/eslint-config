# eslint-config

My ESLint config.

## Install

```sh
npm install --save-dev @1000ch/eslint-config
```

## Usage

In `.eslintrc`:

```json
{
  "extends": ["@1000ch"]
}
```

In `package.json`:

```json
{
  "eslintConfig": {
    "extends": ["@1000ch"]
  }
}
```
